# task-app
タスク管理アプリ

# 1. プロジェクト概要

複数人でタスクを共有、管理するアプリ
友人同士で旅行などの計画を立てるときに必要なものを登録する

# 2. 機能一覧

## できること
- タスクの登録、編集、削除
- ログイン機能
- 権限設定(管理者・一般)
- 1つのグループに所属するメンバー全員でタスクを共有する
- タスクのステータス管理(未着手・着手・完了)
- タスクの区分ごとにフィルタ

# 3. データ定義

## タスク
| 項目 | 説明 |
|---|---|
| タイトル | どのようなタスクか登録できる |
| 説明 | タスクの内容を登録できる |
| ステータス | 未着手・着手・完了で登録できる |
| 期限 | 〇〇日までなどが登録できる |
| 優先度 | 小・中・高 で登録できる|
| 区分 | カテゴリが登録できる(自由入力) |
| 担当者 | タスクを行う担当者を登録できる |
| 登録者 | タスクを登録した人 |

## 権限
| 機能 | 管理者 | 一般 |
|---|---|---|
| タスク登録 | 〇 | 〇 |
| タスク編集 | 〇 | 〇(自分が登録したもの) |
| タスク削除 | 〇 | 〇(自分が登録したもの) |
| ユーザ登録 | 〇 | × |
| ユーザ編集 | 〇 | × |
| ユーザ削除 | 〇 | × |

## ユーザ
| 項目 | 説明 |
|---|---|
| ID | 一意のID |
| パスワード | 8文字以上32文字以内のパスワード |
| 名前 | 対象のIDでログインした人の名前 |
| 権限 | 管理者か一般の権限 |

# 4. 技術スタック

| レイヤー          | 技術スタック|
| ------------- | -------------------------------------- |
| Backend| Java 21 + Spring Boot 3.x + Spring Security|
| API| REST API（OpenAPI / Swagger で定義）|
| Frontend| React（Vite + TypeScript）|
| DB| PostgreSQL（Docker で起動）|
| ORM| Spring Data JPA / Flyway（マイグレーション管理）|
| コンテナ| Docker Compose（API + DB + Frontend）|
| Lint / Format | Backend: Checkstyle + Spotless|
| CI/CD| GitHub Actions|
| IaC| Terraform（EC2）|
| コード管理| GitHub|


# 5. 画面一覧

- ログイン画面
  - ログインを行う
- タスク一覧
  - タスク一覧を確認できる
  - タスクの登録・削除ができる
- フィルタを行う
- タスク詳細
  - タスクの編集
  - タスクの確認
- ユーザ登録、編集、削除画面
  - 管理者のみユーザの登録、編集、削除ができる
